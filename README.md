# 📄 Resume Screening & Candidate Ranking System



---

##  Overview
This project is a Machine Learning–based system that automatically screens and ranks resumes based on a given job description using NLP techniques.

It helps recruiters:
- Shortlist candidates faster  
- Match skills with job requirements  
- Identify missing skills  
- Reduce manual effort  

---

##  Features
✔ Resume text cleaning & preprocessing  
✔ Skill extraction using NLP  
✔ Job description parsing  
✔ Resume-to-job similarity scoring  
✔ Candidate ranking based on role fit  
✔ Skill gap identification  

---

##  Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- NLP (spaCy / NLTK)  
- TF-IDF Vectorizer  
- Cosine Similarity  
- Streamlit (for web app)

---

##  How It Works

1. Load resume dataset  
2. Clean and preprocess text  
3. Extract skills from resumes  
4. Convert text into vectors using TF-IDF  
5. Compare resumes with job description  
6. Rank candidates using cosine similarity  
7. Identify missing skills  

---

##  Output
- Ranked list of candidates  
- Matching score for each resume  
- Extracted skills  
- Missing skills
